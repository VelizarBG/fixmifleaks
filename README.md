# Fix MIF Leaks

Mod that fixes some memory leaks in mods [MI: Foundation](https://www.curseforge.com/minecraft/modpacks/mif) uses.  
Use the mod on both the client and the server for best results.

List of leaky mods:
- [AzureLib](https://www.curseforge.com/minecraft/mc-mods/azurelib)
- [bad packets](https://www.curseforge.com/minecraft/mc-mods/badpackets)
- [Dynamic Surroundings: Remastered Fabric Edition](https://www.curseforge.com/minecraft/mc-mods/dynamic-surroundings-remastered-fabric-edition)
- [Trinkets](https://www.curseforge.com/minecraft/mc-mods/trinkets)
- [Entity Model Features](https://www.curseforge.com/minecraft/mc-mods/entity-model-features)
- [Entity Texture Features](https://www.curseforge.com/minecraft/mc-mods/entity-texture-features-fabric)
- [GeckoLib](https://www.curseforge.com/minecraft/mc-mods/geckolib)
- [Things](https://www.curseforge.com/minecraft/mc-mods/things-fabric-back-ported-some-features)
